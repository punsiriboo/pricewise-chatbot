## การติดตั้ง uv

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
uv sync
```

## การใช้งานพื้นฐาน

### การติดตั้ง package
```bash
uv pip install package_name
```

### การติดตั้ง package จาก requirements.txt
```bash
uv pip install -r requirements.txt
```
